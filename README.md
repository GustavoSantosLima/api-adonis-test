# API VinúBolão

API do projeto VinúBolão

## Executando o projeto

```bash
npm install or yarn
```

Copiar a `.env.example` para `.env`

```bash
cp .env.example .env
```

Para gerar a `application:key`

```bash
adonis key:generate
```

Para rodar todas as migrações

```bash
adonis migration:run
```
